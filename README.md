🎬 Movie App

A modern movie discovery application built with React, Vite, and Tailwind CSS, powered by the TMDB API. Browse trending movies, search for your favorite titles, and explore detailed movie information in a fast and responsive interface.

🚀 Features
🔍 Search movies by title
🎥 View trending and popular movies
📄 Detailed movie information
⭐ Movie ratings and release dates
📱 Fully responsive design
⚡ Fast performance with Vite
🎨 Modern UI using Tailwind CSS
🌐 Real-time data from TMDB API

🛠️ Tech Stack
React
Vite
Tailwind CSS
TMDB API
JavaScript (ES6+)
